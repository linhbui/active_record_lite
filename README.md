active_record_lite
==================

Rebuilding active record
Features:
- attr_accessor
- SQL objects: ::all, ::find, ::where, #insert, #update, #save, ::table_name, ::table_name=, ::columns
- belongs_to, has_many, has_many_through, has_one_through

Topics:
- Database Migration
- Metaprogramming: define_method
- send
